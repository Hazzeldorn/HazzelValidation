Quick start

Requirements

All you need is a webserver supporting PHP Version >= 7.1.

Installation

With composer

HazzelValidation will also work with Composer in the future. However, the library is not yet available on Packagist.

Manually

Simply download the HazzelValidation library, extract the zip folder and include it anywhere in your web project.

Download HazzelValidation

// initialize the framework
require_once( YOUR_APP_PATH . '/HazzelValidation/autoload.php' );

?>

Code example

Data import validation

Here is an example of how imported data can be checked in a few rows using the method chaining approach.

Scenario: A CSV parser function returns an array "input" for each imported row, which contains contact data that now needs to be checked. 

<?php
require_once( YOUR_APP_PATH . '/HazzelValidation/autoload.php' );
use HazzelValidation\Validation as Validation; // you might use an abbreviation such as 'v'

// given array e.g. from csv import function
$input = parseCSV();

if(
    // required fields
    Validation::string()->oneOf(['male', 'female'])->check( $input['gender'] )
    && Validation::string()->notEmpty()->check( $input['firstname'] )
    && Validation::string()->notEmpty()->check( $input['lastname'] )
    && Validation::string()->email()->check( $input['email'] )

    // optional fields
    && Validation::string()->optional()->check( $input['street'] )
    && Validation::string()->optional()->check( $input['city'] )
    && Validation::number()->optional()->length(5)->check( $input['zip_code'] )
){
    // data is valid -> do something
} else {
    // at least one field is not valid;
}
?>
